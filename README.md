# Trabalho de Tabela Hash - Estrutura de Dados

## Como Executar
1. Compile o arquivo: `javac Main.java`
2. Execute: `java Main`
3. O resultado sairá no console em formato CSV.

## Metodologia
O experimento comparou três funções de hashing (Divisão, Multiplicação e Dobramento) sob diferentes fatores de carga.
**Distribuições mais uniformes reduzem o custo médio no encadeamento separado.**
